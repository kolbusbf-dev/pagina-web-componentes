Introducción

Una computadora está formada por diferentes componentes que trabajan juntos para procesar información. En esta página web describimos los principales componentes de una computadora, su función y ejemplos.

Componentes principales

Componente Descripción Imagen
CPU (Procesador) Es el cerebro de la computadora. Se encarga de ejecutar instrucciones y procesar datos.
Memoria RAM Almacena temporalmente los datos y programas que se están utilizando.
Disco duro / SSD Guarda la información de forma permanente, como el sistema operativo y archivos.
Placa base Es la tarjeta principal donde se conectan todos los componentes.
Fuente de alimentación Suministra energía eléctrica a todos los componentes del ordenador.

Componentes de entrada y salida

Dispositivo Función
Teclado Permite introducir texto y comandos.
Ratón Permite interactuar con los elementos de la pantalla.
Monitor Muestra la información visual.
Impresora Permite obtener documentos en papel.
